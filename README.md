# First passage times of Ornstein Uhlenbeck processes to moving boundaries
This repository we simulate the classical Ornstein-Uhlenbeck process and evaluate its first passage time over a barrier numerically, and use a result from my published paper (Link: https://www.cambridge.org/core/journals/journal-of-applied-probability/article/abs/boundary-crossing-problems-and-functional-transformations-for-ornsteinuhlenbeck-processes/3130B53D1870CB723686D1A88CD9DCCA) to find the new density for a moving boundary that is a transformation of the original barrier.
